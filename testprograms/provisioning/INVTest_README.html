Introduction:

This README provides details on how to run the INVTest.java file
Please go through the explanation given to understand the use of the tool.

This document can be divided into two main section:
1. Usage and Valid Arguments
2. TestCases

1.Usage and Valid Arguments

USAGE: java INVTest hostName port methodName
 
S.No Arguments Description Values
1 hostName Name of the machine where NMS is running String
2 port port int 
3 methodName the method (function) that has to be executed String
(valid values are given below)

Please find below the list of all values that can be given for the
third argument methodName.

  • addInventory
  • getInventory
  • updateInventory
  • deleteInventory
  • addVariant
  • getVariant
  • updateVariant
  • deleteVariant
  • getCompleteInventoryList
  • getCompleteVariantList
  • getInventoryNamesWithProps
  • EmsInventory
back
