WebNMS Framework is the leading network management framework that enables telecom equipment vendors and service providers to deliver superior network management solutions in a cost-effective, low-risk, and high-productivity model. You can customize, rebrand, and package WebNMS Framework as a comprehensive Element Management System (EMS) or Network Management System (NMS) application. The scalable architecture of WebNMS Framework makes it possible to manage all networks ranging from small to large distributed networks.

WebNMS Framework is absolutely free for evaluation for a period of 45 days. Please refer to the Installation Guide to proceed with the installation and to get started with WebNMS Framework. Product Features

- Highly customizable framework for creating network management applications such as EMS, NMS, and Mediation solutions.
- Complete FCAPS functionality support - Fault, Configuration, Performance, Inventory, and Security Management
- In-built fail-over and high availability features to help realize 5-9's availability
- Highly customizable and user friendly Java/HTML clients
- Google maps integrated topology view
- Template-based Device Configuration and Provisioning
- Complete choice of Southbound and Northbound protocols including SNMP v1,v2,v3, SSH1, SSH2, HTTP, HTTPS, TL1, and CLI
- Support for various OS (Windows, Linux, and Solaris), DB (Oracle, MySQL, MSSQL, Sybase, Postgres, etc.,) and Hardware platforms
