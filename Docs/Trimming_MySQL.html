Trimming MySQL for WebN MS
After installing the setup.exe of MySQL 4.0.17, here's the procedure to follow:

Start the MySQL Daemon Using mysql\bin\mysqld-nt.exe
    • ./bin/mysqld-nt.exe --no-defaults --base-dir=.  (Also in <NMS_Home>\bin\startMySQL.bat, instead of mysqld call mysqld-nt.exe )
  • Start the MySQL client;
  • use mysql;
  • grant all privileges on *.* to 'root'@'%';//This is optional; needed only for complete permission, Could be customized based on user based access
  • flush privileges;
  • quit
