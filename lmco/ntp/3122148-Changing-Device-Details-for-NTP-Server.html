Please download the zip file using the link and follow the steps stated below to achieve your requirement: (Kindly do backup before proceeding).
http://bonitas2.zohocorp.com/zipUploads/2015_11_02_19_44_08_o_19dsfqq3sllu13n9mei1fhq5u1.tar.gz

1. Edit ApplicationDiscovery.xml file, which is present under <WebNMS_Home>/conf/application file directory and add the highlighted text alone as given below.
 <RESOURCE-TYPE ID="NTP-server" topologyClass="com.adventnet.nms.appln.ntp.discovery.server.model.NtpServer">
        <TEST-GROUP CONDITION="all" TestGroupID="Discovery Rule">
      <RESOURCE-TEST>
        <APPLN-SERVER-TEST objID="Ntp-Test">
          <COMMON-ARGS className="com.adventnet.nms.appln.ntp.discovery.server.NtpDiscovery" timeout="5" >
                        <PORT number="123"/>
                </COMMON-ARGS>
        </APPLN-SERVER-TEST>
      </RESOURCE-TEST>
        </TEST-GROUP>
  </RESOURCE-TYPE> 
2. Edit hibernate.cfg.xml file, which is present under <WebNMS_Home>/classes/hbnlib file directory and add the text given below.
<mapping resource="com/adventnet/nms/appln/ntp/discovery/server/model/NtpServer.hbm.xml"/>
3. Extract NtpServer.zip under <WebNMS_Home>/classes/hbnlib file directory and verify NtpServer.hbm.xml is present in this file path.
            <WebNMS_Home>/classes/hbnlib/com/adventnet/nms/appln/ntp/discovery/server/model/NtpServer.hbm.xml
4. Download and place NtpServerPropertyForm.xml under <WebNMS_Home>/webclient/topo/conf file directory.
5. Add the following lines in EnglishToNative.properties which is present under <WebNMS_Home>/html file directory.
webclient.topo.portNo = Port No
webclient.topo.targetAddress = Target Address 
6. Download NtpServerClasses.jar and place this jar under NmsServerClasses or provide this entry in the classpath of the startnms.sh.

7. Discover ntp device and verify the results.

We also attach all the source files and xml files (NtpFiles.zip) for reference.
