Please note that we had not got any specific date for the release - As of its not scheduled by the end of July (Sorry for the typo yesterday).
The main features in this build are:
* Updated WebClient UI
* Key Performance Indicators(KPI)
* Decimal Support in Performance module
* Netconf protocol support
* Dynamic Update feature for topology, and fault modules in WebClient
* Support for Tomcat 7, Oracle & OpenJDK, MySQL 5.6.x support & postgres latest versions.

For your convenience, we had got an early access.
The early access build that is compatible with WebNMS 5.2 is available in the below URL.
https://bonitas.zohocorp.com/232530/webnms5/05Apr2013/WebNMS_Framework_5_2_SP-0_08.ppm

Please follow the steps below to install and work with this build:

Please install WebNMS 5.2 build.

Install the PPM over this build using the Update Manager tool present in the <WebNMS Home>/bin/ directory. (If you face any disc space issue in Windows, please replace lib/DiscSpace.dllfrom here)

Now unzip the below ZIP file in the WebNMS Home directory. This contains configurations files required for this build. 
https://bonitas.zohocorp.com/232530/webnms5/15Feb2013/conffiles.zip

Run the compileJSP.sh / bat script file to compile the JSP files. (This step is required for the early access build).

Start WebNMS.

Do get back to us if you need any clarifications on this early access ppm.
