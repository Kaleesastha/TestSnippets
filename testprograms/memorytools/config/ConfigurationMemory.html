Operation Description
Creating a task Create a task with say 100 attributes defined.
Executing a task  1. Execute a task over say 100 devices
2. Execute a template task over 100 devices.
Devicelist Create a devicelist containing say, 100 devices, associate this devicelist with a task and execute.
Applying rollback Execute a task (with some 100 attributes) such that it goes for rollback (for multiple devices) 
Simultaneous task execution Execute the same task from multiple clients (say 5) simultaneously
Executing a combined task  Create a combined task spread across different protocols. Execute the same over multiple devicelists
Task upload Set very large values onto an SNMP agent. Try to upload the same to a task
