Please find the below procedure to get the handle of the server closed dialog.

Write a class by extending JOptionPane and overwrite the createDialog(Componet, String) method.
Write a class by extending AbstractBaseNmsPanel and provide this class name in NmsPanels.conf file present under $NMS_HOME/conf directory as follows:
<PANEL className="test.TestPanel" PANEL-KEY="TestPanel"/>

In init(JApplet) method of this class, you need to create your extended JOptionPane instance and set as close dialog in MainSocketClient by using the showCustomizedServerClosedDialog(JComponent, String) method.

Note : The MainSocketClient class instance in WebNMS Client can be accessed by using PureClientUtils.commonSocket variable.

By doing this, whenever the server shut down, your extended JOptionPane class's createDialog(Component, String) method will be called.
Inside this method you can write your implementations and return the super value.

Please refer the uploaded zip (http://bonitas.zohocorp.com/4264249/webnms4/13Dec2011/ClientCloseDialog.zip) which contains TempDialog.java and TestPanel.java & its class files for details.
