To silently install WebNMS

Execute the command (optional parameters):

<Absolute_Path_of_binary/exe> -is:javaconsole -is:javahome yourjavapath/jre -is:log log.txt -is:tempdir <New_Temp_Directory> -silent -options <absolute_path_if_installoptions.txt> -P ProductRoot.installLocation=<Install_Location>
Where installoptions.txt is the following file:

#******************************************************
-W locale.lang=English
-W OEMProducts.selectedValue=OEMADV
-W OEMAdvance.selectedValue=OEMADVBE
#The above can be changed as OEMADVFE or OEMADVClient accordingly
-W StdCustom.optComponents=Javadocs,Tutorials
#The above can be changed as None if you do not want Tutorials/Javadocs
-G replaceExistingResponse="yesToAll"
#*******************************************************

To silently install a service pack:
sh UpdateManager.sh -u conf -c -option i -ppmPath <Absolute_Path_of_PPM> -h <Absolute_Path_of_NMS_Home> -DJAVAHOME=<JDK_Home> -DLANGUAGE=English

If you do not want to compile JSPs, configure -DJAVAHOME=NONE

Command Line installation of Nar:
Execute the following commands in the command prompt (from $NMS_Client_home/bin/developertools )

Command Line installation of Nar:
 * CommandLineStudioNarInstaller.bat <Absolute_path_of_the_nar>
Eg: CommandLineStudioNarInstaller.bat C:\WebNMS\DiamondTest03__21.0.narThis installs the Nar in the client

Command Line un-installation of Nar:
* CommandLineStudioNarUnInstaller.bat <Name_of_the_nar>
Eg: CommandLineStudioNarUnInstaller.bat DiamondTest03__21.0
(Name got from NetMonitor/build/uninstall.xml (<INSTALLED-NARS APP_VER="DiamondTest03__21.0" NAME="DiamondTest03__21.0" TYPE="PWSAPPLICATION">)
This uninstalls the Nar in the client.

 If you choose to install a standalone client, please follow the below steps to make the above scripts work:

1. Create $NMS_Client_home/bin/developertools directory in the Standalone client
2. Copy CommandLineStudioNarInstaller.bat & CommandLineStudioNarUnInstaller.bat to $NMS_Client_home/bin/developertools
3. Copy NmsServerClasses.jar to $NMS_Client_home/classes (all these files from BE server)

Now execute the above commands.
