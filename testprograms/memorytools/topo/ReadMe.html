README FOR VARIOUS FILES
 

1. AddNetwork.java
     Module: TOPO

        This adds the the Network object and deletes the same object in an indefinite loop using TopoAPI. It takes the "hostname" to lookup the TopoAPI and then the "Network Address"  which is to be added and deleted.The same can be invoked using the addNetwork.sh file which calls the class.

 Usage:  java AddNetwork <hostName>  <Network Address> .
 

2.  MyAlertClient.java
      Module: SAS
 
         This starts the Application Client and creates an Custom View and Removes the same.This can be invoked using the "myAlertClient.sh" file which invokes the class.The host,web-server port, user name and password needs to edited in the "MyAlertClient.java" file.

 Usage:   java  MyAlertClient 
 

3. AddObjects.java
    Module: TOPO

        It adds the ManagedObjects to the database using the TopoAPI.

    Usage:  java AddObjects
 

4.  TopoTest.java
     Module: TOPO

        This adds,delets and updates the object in different order in different thread.

    Usage: java TopoTest <hostName>

5.   TopoNotification.java
      Module: TOPO

           This class registers ifself as various observers and deregisters.To check if the observers are getting registered and deregistered properly.

    Usage:  java TopoNotification <host>
 
6.   MapAPITest.java
      Module:  MAP

            All the MAP Api methods can be invoked through this.
