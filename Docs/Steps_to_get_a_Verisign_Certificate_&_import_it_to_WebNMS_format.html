The steps to follow are:

Create the BEServer.key:   openssl genrsa -rand world.png -out BEServer.key 1024

Create the BEServer.csr:    openssl req -new -key BEServer.key -out BEServer.csr

In Verisign Inc, give the same inputs (as that of BEServer.csr) & now a mail will arrive in your Inbox:
There will also be a certificate ( a random no.) at the end of the mail. Do not use that. Instead go to the link given in the mail (It will be like http://www.verisign.com/server/trial/faq/index.html)
And then click the Secure Site Trial Root CA Certificate >> link. You will be shown the cerificate (again a random no. compliant to the X509 format) Copy this & save the same as BEServer.cer

Now import the same to Truststore: keytool -import -alias bessl -keystore Truststore.truststore -file BEServer.cer

Create the BETranskey.key: keytool -genkey -alias betransssl -validity 3000 -keystore BETransKey.key -storepass webnms -keypass webnms

Create the BETranskey.cer: keytool -export -alias betransssl -keystore BETransKey.key -file BETransCer.cer
Import the BETranskey.cer to Truststore: keytool -import -alias betransssl -keystore Truststore.truststore -file BETransCer.cer
