getDate()

Description: To convert the long value to simple date format ( MMM dd,yyyy hh:mm:ss a ) and using local system's timezone preference.

usage2:

java com.adventnet.nms.webclient.utils.GenericTestUtil   getDate <arg1: long value of the date> <arg2: Date Format default: MMM dd,yyyy hh:mm:ss a>

Description: To convery the long value to with format specified in arg2 and using local system's timezone preference.

generateAllSeverityEvents()

usage1:

java com.adventnet.nms.webclient.utils.GenericTestUtil generateAllSeverityEvents <entity name>

Description: This method will create events with severity mentioned SeverityInfo.conf. By default, this conf will have all the predefined severities. So if you are giving entity name as "rameshkumarp", then the following events will be created,

Failure Object (Entity Name)
 Severity
rameshkumarp_Critical_Event
 Critical
rameshkumarp_Major_Event Major
rameshkumarp_Minor_Event Minor
rameshkumarp_Warning_Event Warning
rameshkumarp_clear_Event clear
rameshkumarp_info_Event info
rameshkumarp_unknown_Event unknown
  

generateClearAlerts()

usage1:

java com.adventnet.nms.webclient.utils.GenericTestUtil generateAllSeverityAlarms <entity name 1 > <entity name 2>

Description: This method will inturn call generateAllSeverityEvents() to generate events with all severities which inturn will make the corresponding alarms to be added in to the database. For clear alarm, the entity name will be <entity name 2>_Critical_Event.
