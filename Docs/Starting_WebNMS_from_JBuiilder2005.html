Starting AdventNet Web NMS server in JBuilder 2005

 Before starting Web NMS Server in JBuilder LD_LIBRARY_PATH has to be exported from the console from which the JBuilder is started. Export LD_LIBRARY_PATH as follows.

LD_LIBRARY_PATH=$LD_LIBRARY_PATH:<Web NMS Home>/lib:<Web NMS Home>/apache/lib
export LD_LIBRARY_PATH

In Windows just setting the LD_LIBRARY_PATH is not needed

For starting Web NMS Server in JBuilder, the following steps are to be followed.

  • Create a new project in JBuilder 2005.

  • On the Project menu, click Project Properties.

  • Select the Paths tab and then select Required Libraries tab.

  • Click the Add button.

  • On the tree that appears, select User Home and then click the New button.

  • Type a name for your library and click the Add button to add the library paths.

  • Browse and add the following directories and jar files to the list. You can select multiple items by pressing Ctrl key.

  • Save the project.

  • On the Project menu, click Project Properties.

  • Select the Run tab and then click the New Button available in the right side.

  • On the dialog that appears select the application tab.

  • Click the Browse button (the button with "...") to <Web NMS Home>/classes/com/adventnet/launcher/nms and set the StartNmsJdbc as Main Class.

  • Copy and Paste the following in the VM Parameters textfield.

-Dwebnms.rootdir=<Web NMS Home>/
-Djava.library.path=<Web NMS Home>/lib

-Djava.rmi.server.codebase="http://localhost:9090/classes/

http://localhost:9090/classes/ManagementServer.jar

http://localhost:9090/classes/AdventNetTftp.jar http://localhost:9090/classes/AdventNetCLI.jar

http://localhost:9090/classes/AdventNetSnmp.jar http://localhost:9090/AdventNetSNMPv3/classes/

http://localhost:9090/classes/AdventNetJmxAgent.jar

http://localhost:9090/classes/xmojo.jar
http://localhost:9090/classes/jta.jar" -mx100m

Enter these fields such that there is a single space between each arguments.

  • Enter the following in the Application Parameters textfield.

           APACHE_DIR <WebNMS Home>/apache WEBSERVER_PORT 9090 WEBCONTAINER_PORT 8009 TOMCAT_SHUTDOWNPORT 8005 TOMCAT_DIR <WebNMSHome>apache/tomcat ROOT_DIR <WebNMS Home>/

  • Run the Project. You can run the Project in Debug Mode also.

Starting Web NMS Client in JBuilder 5

  • Create a new project in JBuilder.

  • On the Project menu, click Project Properties.

  • Select the Paths tab and then select Required Libraries tab.

  • Click the Add button.

  • On the tree that appears, select User Home and then click the New button.

  • Type a name for your library and click the Add button to add the library paths.

  • Browse and add the following listed directories and jar files to the list. You can select multiple items by pressing ctrl key.

<WebNMS Home>/classes;

<WebNMS Home>/NetMonitor/build/ConfigPanel.jar

<WebNMS Home>/classes/jfreechart.jar

 <WebNMS Home>/classes/jcommon.jar

  • Save the Project.

  • Click Project Properties in the Project menu.

  • Select the Run tab and then click the Application tab.

  • Click the Browse (...) button to <WebNMS Home>/classes/com/adventnet/nms/startclient and set the WebNMSClient as Main Class.

  • If the WebNMS server is running in the same machine, enter localhost 9090 in Application Parameters textfield. If it is running in a different machine, specify the host name of that machine instead of localhost.

  • Run the Web NMS Client project.

Ensure that the Web NMS Server is started before running the client.
