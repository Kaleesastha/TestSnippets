Do you have a license file? (solid.lic)
If yes, here is what we do to install solid as a windows service:

D:\CVS\framework\db\bin\solid.exe -sinstall,"ConSentry InSight Database", D:\CVS\framework\db\bin\solid.exe,autostart

We call the above command from D:\CVS\framework\db\data directory. This takes care of 2 things. One, the solid.lic file is present in this directory. Two, when solid gets installed as a windows service, it executes in this directory making sure the license file is found.

If you open the services window and right click on solid service (ConSentry InSight Database), and select Properties, a window will pop up. Under the general tab, there is a non editable text field called "Path to executable". Here is what it reads like for us after we do the above step:

D:\CVS\framework\db\bin\solid.exe -c "D:\CVS\framework\db\data" -n "ConSentry InSight Database" -s start
