We have given herewith the steps to compile the apache for the Solaris / Linux OS.

Ensure beforehand to make, gcc, ar, autoconf, libtool, perl (>5), automake packages are there in the system & the system PATH variable contain the following:
Download httpd-2.2.8.tar.gz or httpd-2.2.6.tar.gz from http://archive.apache.org/dist/httpd/ and issue the following compounds:

$ gunzip  httpd-2.2.6.tar.gz
$ tar xvfi httpd-2.2.6.tar
$ export PATH=/usr/local/ssl:/etc/usr/ccs:/usr/bin:/usr/local/bin:/usr/ucb:/etc:/etc/usr/ccs:/usr/bin:/opt/sfw/share:/opt/sfw/bin:/usr/ccs/bin:$PATH
$ cd <Directory in which the httpd-2.0.47 is created>

For non-ssl
$ ./configure --enable-rewrite=shared --prefix=<APACHE2_HOME>
$ make
$ make install

To Install OpenSSL:
Download OpenSSL from http://www.openssl.org/source/openssl-0.9.7b.tar.gz 
After unzipping openssl-0.9.7b.tar.gz follow the below commands as root user from <Dir_in_which_OpenSSL_is_present>:

# ./configure
# make
# make install

And installing Apache in SSL Mode:
$ ./configure --enable-ssl --with-ssl="OpenSSL_HOME" --with-egd-pool=/var/run/egd-pool --disable-threads --enable-rewrite=shared --prefix="APACHE2_HOME"
$ make
$ make install
