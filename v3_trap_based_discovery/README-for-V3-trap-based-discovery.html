Here're the steps for a v3-trap-based-discovery:

1.You need to know the networks from which V3 traps may be received in WebNMS trap port (162 by default).
Based on the network IPs you need to populate the UserTable.

2. You can use this piece of code snippet to populate the USERTABLE.
You can modify this code to call only PopulateUserTable.insertIntoUserTable and remove all other calls and add this as a process in NmsProcessesBE.conf.

3. Download V3CredentialsValidator.java & V3CredentialsValidatorImpl.java, compile the same and add it to classpath.
Note that V3CredentialsValidatorImpl.java is an implementation class (which is open sourced) and you can change the same based on your need.

4. Append nmsInterfaces.conf with V3CredentialsValidator="test.V3CredentialsValidatorImpl" as shown in this file.

5. Now you can restart the server.

6. Once you receive a v3-trap from a not-yet-discovered-IP (whose network credentials are already populated in UserTable), then that trap will trigger the discovery of that node as well as network.
